# goit-js-hw-05

**Автор:** Максим Канюка

## Опис

Домашнє завдання №5 з курсу JavaScript від GoIT.

## Завдання

- **task-1.js** — Імена користувачів: стрілочна функція `getUserNames(users)`,
  що використовує `map()` та повертає масив імен користувачів.
- **task-2.js** — Користувачі з другом: стрілочна функція
  `getUsersWithFriend(users, friendName)`, що через `filter()` та `includes()`
  повертає користувачів, у яких є друг із заданим іменем.
- **task-3.js** — Сортування за кількістю друзів: стрілочна функція
  `sortByDescendingFriendCount(users)`, що через `toSorted()` повертає новий
  масив користувачів, відсортованих за спаданням кількості друзів.
- **task-4.js** — Загальний баланс: стрілочна функція
  `getTotalBalanceByGender(users, gender)`, що ланцюжком `filter().reduce()`
  повертає загальний баланс користувачів обраної статі.
